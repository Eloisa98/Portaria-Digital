# 🏢 Portaria Digital - Entrega Fácil

**Portaria Digital** é um aplicativo desenvolvido para simplificar o processo de gerenciamento de encomendas em condomínios. O app permite que a portaria registre encomendas recebidas e notifique os moradores quando suas entregas estiverem disponíveis para retirada. Com uma interface amigável e funcionalidades práticas, o objetivo é otimizar a comunicação entre a portaria e os moradores, garantindo que as encomendas sejam geridas de forma eficiente.

## 📖 História

O projeto surgiu da necessidade de melhorar a gestão de encomendas em condomínios, onde muitas vezes os moradores não são informados sobre a chegada de suas entregas. A ideia é proporcionar um sistema que centralize as informações das encomendas, permitindo que a portaria registre e notifique os moradores de maneira prática e rápida. Com isso, buscamos reduzir o tempo de espera e aumentar a satisfação dos moradores.

## ✅ Funcionalidades

| Funcionalidade                | Descrição                                                                   |
|-------------------------------|---------------------------------------------------------------------------|
| 📦 Registro de Encomendas     | A portaria pode registrar as encomendas recebidas com nome e data.       |
| 🔔 Notificações               | Os moradores recebem notificações quando suas encomendas estão disponíveis para retirada. |
| 📊 Gerenciamento de Status     | O status das encomendas pode ser atualizado (ex: "aguardando retirada", "retirada"). |
| 💻 Interface Amigável          | Design intuitivo para facilitar o uso tanto pela portaria quanto pelos moradores. |

## 📊 Fluxograma do Sistema

![Fluxograma-Portaria-Digital](https://github.com/user-attachments/assets/125a0b4e-bc0d-421d-a09b-12c0d5710786)

**Etapas do Fluxograma:**

1. 📦 **Encomenda**: A portaria recebe a encomenda do entregador.
2. 📝 **Registrar Encomenda**: A portaria registra as informações da encomenda no sistema.
3. 👤 **Associar Encomenda ao Morador**: O sistema associa a encomenda ao morador correspondente.
4. 🔔 **Enviar Notificação ao Morador**: O sistema envia uma notificação ao morador.
5. ⏳ **Aguardando Retirada**: A encomenda fica registrada como "aguardando retirada".
